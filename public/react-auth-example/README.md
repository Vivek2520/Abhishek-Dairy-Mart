# React Auth Example

This folder contains a drop-in React Router + AuthContext setup for:
- login
- signup
- protected routes
- logout/session handling

Use these files in your React app source directory:
1. `AuthContext.jsx`
2. `ProtectedRoute.jsx`
3. `AppRoutes.jsx`
4. `pages/*`

Make sure your React app points API calls to the same backend (`/api/...`).
