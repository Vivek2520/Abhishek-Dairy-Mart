# Project Name

A full-stack web application with Node.js backend and React frontend.

## Structure

- `backend/` - Node.js/Express server
- `frontend/` - React application
- `scripts/` - Utility scripts
- `database/` - Database configurations and seeds
- `docs/` - Documentation

## Setup

### Backend

cd backend
npm install
npm start

### Frontend

cd frontend
npm install
npm start
